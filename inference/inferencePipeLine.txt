    │    FP32 input → FP32 1×1 conv → FP32
    │
    ├─── Conv2 path:
    │    INT8 passthrough ← (from conv1, NO requantize!)
    │    → pad INT8 (just memcpy, no absmax)
    │    → INT8 im2col → VNNI interleave (u8)
    │    → vpdpbusd (u8 × s8 → s32)
    │    → dequant s32 → FP32 (apply fused_scale, fused_bias)
    │    → ADD FP32 residual (shortcut or identity)
    │    → ReLU
    │    → FP32 output
    │
